To filter your search results

- Type +.
- Select a filter field from the dropdown.
- Select a filtering term from the suggestion list or type your own.

As a shortcut you can type the filter field followed by a colon, e.g. location:.

To exclude results containing the terms

As above but type - to create an 'excluding' filter.

For existing filters, press + on the left hand side to exclude the term.
