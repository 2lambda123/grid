How to search

• Results always display photos containing all entered search terms (AND)
• Type - before the term to exclude it from results • Type + to choose from a list of filters, see examples below
• As a shortcut you can type the filter field followed by a colon, e.g. credit:
• For active filters, click + on the left hand side to toggle between including and excluding
• Wrap your search terms in quote marks to search for an exact phrase
