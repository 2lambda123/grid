Upsert image via projection

This option will run a projection against the given media ID, and OVERWRITE any currently stored image data in Elasticsearch. (ie. metadata, usage rights, leases etc. The image file itself will not be touched)

This is a potentially destructive operation, so be sure to check the result of projection before running!

If you're happy, then enter the media ID below and hit 'submit'...
